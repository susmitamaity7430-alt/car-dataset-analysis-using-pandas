🚗 Cars Data Cleaning & Analysis using Pandas

📌 Project Overview

In this project, I worked with a cars dataset taken from Kaggle.
The main focus of this project is data cleaning and basic analysis using Pandas.

---

📂 Dataset

- Source: Kaggle
- Type: Cars dataset
- Contains information like car name, fuel type, price, etc.

---

🛠️ Tools Used

- Python
- Pandas
- Jupyter Notebook

---

🔍 Work Done

1. Data Loading

- Loaded dataset using Pandas ("read_csv")

2. Data Understanding

- Checked dataset structure using:
  - ".info()"
  - ".describe()"

3. Data Cleaning

- Checked missing values using "isnull()"
- Removed or handled null values
- Removed unwanted or duplicate data
- Filtered required data using conditions

4. Data Analysis

- Used "value_counts()" to understand data distribution
- Applied filtering to analyze specific data
- Explored different columns and their relationships

---

📊 Key Learnings

- How to clean real-world datasets
- Handling missing values
- Filtering and selecting data
- Understanding dataset structure using Pandas

---

▶️ How to Run

1. Open Jupyter Notebook
2. Load the dataset
3. Run the code step by step
